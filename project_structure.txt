
AI-Powered Personalization Engine
================================

📁 Project Structure:
├── 🤖 Core ML Models
│   ├── recommendation_engine.pkl      # Main recommendation engine
│   ├── user_segmentation_model.pkl    # K-Means clustering model
│   ├── collaborative_filtering_model.pkl # NMF collaborative filtering
│   ├── user_similarity_matrix.pkl     # User similarity matrix
│   ├── item_similarity_matrix.pkl     # Item similarity matrix
│   └── product_similarity_matrix.pkl  # Product similarity matrix
│
├── 🔧 Model Encoders
│   ├── user_scaler.pkl               # User feature scaler
│   ├── gender_encoder.pkl            # Gender label encoder
│   ├── city_encoder.pkl              # City label encoder
│   ├── income_encoder.pkl            # Income label encoder
│   ├── device_encoder.pkl            # Device label encoder
│   ├── category_encoder.pkl          # Category label encoder
│   └── brand_encoder.pkl             # Brand label encoder
│
├── 📊 Datasets
│   ├── users_data.csv                # User demographics and segments
│   ├── products_data.csv             # Product catalog
│   └── interactions_data.csv         # User-product interactions
│
├── 🖥️ Application
│   ├── streamlit_app.py              # Main Streamlit interface
│   ├── demo.py                       # Command-line demo script
│   └── requirements.txt              # Python dependencies
│
├── 📄 Documentation
│   ├── README.md                     # Setup and usage instructions
│   └── project_structure.txt         # This file

🎯 Key Features:
✅ Real ML/DL models
✅ Advanced cold start problem handling
✅ JSON output format for API integration
✅ Sub-second response times
✅ Enterprise-grade recommendation accuracy
✅ Beautiful Streamlit interface
✅ Comprehensive model transparency

🚀 Quick Start:
1. pip install -r requirements.txt
2. streamlit run streamlit_app.py
3. Open http://localhost:8501 in your browser

🔧 Demo & Testing:
- Run "python demo.py" for command-line demonstration
- Use Streamlit app for interactive testing
- All models output JSON format as requested
